# ToDoListApp
A beginner-friendly to-do list desktop application built in Python.

## Features
- Add Task, View All Tasks, Mark a Task as Done, Remove a Task
- Beginner-friendly code

## Technologies
- Python

## How to Run
1. Install Python
2. Run `todolist_app.py`
